Show the extra stats and performances of the participating teams of an event.

== Description ==

This plugin is an extension for [SportsPress](https://wordpress.org/plugins/sportspress/). Give your guests the option to see extra stats about the participating teams of an event.

== Installation ==

1.  Install and activate Enhanced Event Stats for SportsPress plugin
2.  Enjoy!
